# Pokedex

## Architecture & Library
* Language : Swift

* Architecture : MVVM(RxSwift)

* UI : code-based SnapKit (storyboardless)

* Database : Realm

* Network & Image : Alamofire, Moya, Kingfisher

* JsonParsing : SwiftyJSON

* OpenAPI : [PokeAPI](https://pokeapi.co/)

* etc : Then
